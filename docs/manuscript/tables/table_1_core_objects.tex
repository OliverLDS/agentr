\texttt{MemorySpec} & R6 class & Reviewable schema for context, semantic, episodic, and procedural memory fields. \\
\texttt{KnowledgeSpec} & R6 class & Curated narrative knowledge, first-class graph knowledge, and future vector-reference metadata. \\
\texttt{WorkflowProposal} & R6 class & Represents one candidate workflow proposal and its lifecycle state during review, discussion, approval, rejection, or supersession. \\

\texttt{MemoryProposalState} & R6 class & Stores the approved memory schema together with candidate memory-schema proposals. \\
\texttt{KnowledgeProposalState} & R6 class & Stores approved narrative knowledge together with curated knowledge proposals. \\
\texttt{KnowledgeGraphProposalState} & R6 class & Stores approved graph knowledge together with graph-node and graph-edge proposals. \\
\texttt{DesignReviewSpec} & R6 class & Packages workflow, memory, knowledge, proposal-state, and feedback-schema sections for standalone HTML review. \\
\texttt{IntelligentAgent} & R6 class & Runtime-oriented wrapper built from an \texttt{AgentSpec}, used to carry selected subsystem configuration and runtime state without defining the execution stack itself. \\
